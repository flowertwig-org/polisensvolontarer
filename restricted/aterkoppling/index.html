Återkoppling

Tack för att du vill förbättra denna webbplats antingen genom förbättringsförslag eller buggrapportering.
Det är tack vare dig som vi ständigt kan förbättra denna webbplats och göra upplevelsen för dig bättre.

Vi använder oss av GitHub som ärendehanteringssystem och för att kunna lämna förbättringsförslag eller buggrapporter till oss behöver du ha ett konto där.
När du klickar på någon av länkarna nedan så har du antingen möjlighet att logga in eller skapa ett konto (Att skapa ett konto är gratis).

Under respektive rubrik nedan hittar du information som gör att den återkoppling du ger oss hjälper på bästa sätt.
Vänligen följ detta format så mycket som möjligt för att göra det möjlig för oss att snabbare och bättre göra de ändring du vill ha.
